# PrimeDash — The Throne

> *"One dashboard. Total command."*

**Role:** Command Center · Visualization Layer · Forgemaster Interface

PrimeDash is the Pantheon's control room — a unified dashboard where every Prime reports its status, every metric is visible, and every command can be issued from a single interface.

## Core Functions
- **Unified Monitoring** — real-time status of all 25 active Primes
- **Revenue Tracking** — live War Chest balance, MidasPrime metabolic cycles
- **Alert Center** — aggregates SentinelPrime alerts, ZeusPrime signals, and system events
- **Command Interface** — issue directives to any Prime from a single control surface

## Pantheon Role
PrimeDash is the Forgemaster's eyes. Ghost Operator mode means zero physical presence — PrimeDash makes the entire empire visible from a single screen.
